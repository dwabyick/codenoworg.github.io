Beginner Level

Projects intended for students who are beginning with Ruby.

Day 1

Basics

Introduce students to the programming tools they’ll be using throughout the program.

  • Cloud 9: A cloud IDE – Get familiar with your development environment.
  • Command Prompt Basics and IRB – Get familiar with the command prompt; learn about using IRB.

Programming Projects (to do in order)

  • Hi-Lo Game – Your first exposure to Ruby, focusing on the most important concepts and syntax elements.
  • Rock, Paper, Scissors – Try your hand at building a more complex game that builds on the concepts learned in Hi-Lo.

Day 2

Programming Projects (to do in order)

  • Calculator– Build a basic calculator that can be used in the terminal.
  • Colorize! – Have some fun with colors in the terminal.

Day 3

Review Projects

  • MadLib Game – Review of string variables and puts/gets
  • Hobbit Game – Review of conditionals and boolean logic

Programming Projects (to do in order)

  • Encryptor – Write a program that enables you to encrypt and decrypt secret messages.
