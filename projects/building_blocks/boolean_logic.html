Boolean Logic

Conditionals and looping routines both contain expressions that can evaluate to “false” or “true”. Also, both can include one expression to be evaluated or a combination of expressions. When using a combination of expressions, we use boolean operators.

The main boolean operations are:

  • AND: denoted by &&
  • OR: denoted by ||
  • NOT: denoted by !

The rules for combining expressions using each boolean operator are as follows:

AND: Notice that both P and Q need to be true for the whole expression to evaluate to true.

P Q P AND Q
T T T
T F F
F T F
F F F

OR: Notice that the whole expression evaluates to true if either P or Q are true, or both P and Q are true.

P Q P OR Q
T T T
T F T
F T T
F F F
P NOT P
T F
F T

Try the following in irb:

  • number is a variable being used to store an integer
    • number = 5
    • puts number
    • number = 10
    • puts number
  • name is a variable being used to store a string
    • name = “CodeNow”
    • puts name
    • name = “CodeLater”
    • puts name

Notice that number, name and puts are all in lower case. And that CodeNow and CodeLater have both lower and upper case letters. Also, they are enclosed in quotations. Capitalization and punctuation are incredibly important in computer programming. Commands in Ruby are always lower case, variables always start with a lowercase letter, and strings are always enclosed in quotations.
