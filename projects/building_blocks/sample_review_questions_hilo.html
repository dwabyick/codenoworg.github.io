Sample Review Questions

At the end of a project you will want to do a quick review with students. Here are some sample review questions and answers you can use:

Programming Practices

  1. Why do we want to write pseudocode? It helps us understand the logic of our program. It acts as a to-do list for our program.
  2. What is the order in which the computer reads your code? Top to bottom
  3. Why do we want to do refactoring? When would we do it? To make our code cleaner and more understandable
  4. Why do we want to write our own methods?

Variables

  1. Why do we want to use variables in our program? To be able to store data and information
  2. What is one variable we used in this program? What type of data did we store in it?
  3. What is the difference between using “=” and “==”? Assignment vs. comparison

Data Types

  1. What is the main difference between strings and integers? Strings store text, integers store numbers
  2. What are some of the operations we can perform on integers? Add, Subtract, multiply, and divide
  3. What are some methods for strings we have learned so far? gets and puts

Control Structures

  1. What are conditionals? When do we use them? Conditionals help us control the flow/execution of the program
  2. What are the key words we used when adding conditionals to our program? If & end
