Arrays

An array holds a collection of items. You can think of it as a list. In Ruby, the types of items in the array is not important. For example, a 3 item array can hold an integer, a string, and another array. The order of items may or may not be important.

Arrays are zero-indexed. Its elements can be accessed by using the bracket operators [n], where n is the index or position of the element we are trying to access. Since this is an introductory workshop, do not worry about this too much. Often convenience methods, such as “first”, “last”, and “sample” are used to access the elements of an array.

In addition, arrays have other methods we can use to find out different things such as "lenght", "size", "include?", and "empty?".

Finally, iterating through an array is something that is commonly done. Iterating through an array is, basically, accesing each element in the array and doing something with it, such as printing it to the screen. To do this, we use the "each" iterator.

Here is an example: names_array.each do |name| puts name end. We can read this stament as "for each name in names_array print the name to the screen.

In essence, the "each" iterator is a special kind of method that behaves like a loop.
