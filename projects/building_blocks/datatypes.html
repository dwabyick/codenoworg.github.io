Datatypes: String and Integers

The data that we store and use in our programs can be of different types. Two main types are string and integer.

Numeric data that are whole numbers are of type integer, hence the variables we use to store it are also of type integer.

Among the operations we can use with integers are the following: addition, subtraction, multiplication, and division.

To perform these operations we use the following symbols: + for addition, - for subtraction, * for multiplication, and / for division.

Textual data is of type string, hence the variables we use to store it are also of type string. Strings are written inside double quotes.

We can also perform “operations” on strings. In order to perform “operations” on strings we use methods. Some of the methods we can use are:

  • puts = when we want to print something to the screen.
  • gets = when we want to capture user input.
  • .chomp = when we need to strip the new line character.
  • .downcase = when we want to convert all characters to lowercase.
  • .upcase = when we want to convert all characters to uppercase.
  • .capitalize = when we want to convert the first character to uppercase and the remainder to lowercase
